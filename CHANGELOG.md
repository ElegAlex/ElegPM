# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-12

### Added
- Initial release of ElegPM
- Complete project management features:
  - Project creation and management with status tracking
  - Hierarchical task organization with subtasks
  - Kanban board view with drag & drop
  - Gantt chart visualization with dependencies
  - Milestone tracking and management
  - Resource allocation and workload management
  - Dashboard with statistics and activity log
- Import/Export capabilities:
  - Excel (.xlsx) import with flexible column mapping
  - PDF export for reports
  - Excel export for data analysis
  - PNG/JPG export for visualizations
- Modern UI inspired by Linear design philosophy
- JSON file-based storage for local data persistence
- Cross-platform support (Windows and Linux)
- Standalone desktop application (no network dependencies)

### Technical Features
- Built with Electron + React + TypeScript
- Tailwind CSS for styling
- Zustand for state management
- Secure IPC communication architecture
- Context isolation and security best practices

### Branding
- Elegartech branding and identity
- About section with links to GitHub, website, and blog
- Professional README and documentation
- MIT License for open-source distribution
